# aula-inova-o

Projeto de exemplo organizado em pastas separadas para HTML, CSS e JavaScript.

## Estrutura do projeto

- `html/`
  - `index.html` — página principal do projeto.
- `css/`
  - `styles.css` — estilos visuais da página.
- `js/`
  - `script.js` — comportamento e lógica do JavaScript.

## Como usar

1. Abra o arquivo `html/index.html` em um navegador.
2. Ou use uma extensão como Live Server no VS Code e acesse `html/index.html`.

## O que o projeto faz

- `index.html` carrega a interface básica.
- `styles.css` aplica o layout e estilo.
- `script.js` adiciona um evento de clique ao botão e mostra uma mensagem atualizada.

## Nota

A estrutura mantém o conteúdo organizado e facilita a manutenção da aplicação.
